# 管理系统登录配置说明

## 1. 配置文件
- 路径：`admin-backend/config/admin.json`
- 结构：
```json
{
  "user": "admin",
  "password": "<bcrypt哈希>"
}
```
- 当前仓库内默认账户：`admin / Qq12345678`（bcrypt 哈希存储）。

## 2. 修改管理员密码
1. 在任意 bcrypt 工具生成新哈希（或联系开发在后端提供修改接口）
2. 将 `password` 替换为新哈希，保存后在管理后台点击“系统 → 热加载”

## 3. 接口说明
- `POST /api/v1/auth/login`：登录，Body: `{user, password}`，返回 `{token, csrf_token, ...}`，并设置 `csrf_token` Cookie（SameSite=Strict）。
- `GET /api/v1/auth/me`：获取当前用户信息（需 `Authorization: Bearer <token>`）
- `POST /api/v1/auth/logout`：登出，清除 CSRF Cookie
- 受保护接口需携带：
  - `Authorization: Bearer <token>`
  - 对于写操作（POST/PUT/PATCH/DELETE）：`X-CSRF-Token` 与 Cookie `csrf_token` 一致

## 4. 安全策略
- 令牌默认有效期 2 小时，可通过环境变量调整：
  - `ADMIN_JWT_TTL`（秒）
  - `ADMIN_JWT_SECRET`（请在生产注入强随机密钥）

## 5. 前端说明
- 登录页面：`/login`
- 登录成功后，令牌与 CSRF Token 保存至 `localStorage`，拦截器自动在请求头注入
- 未登录访问任何管理页面将被重定向至登录页
